# Agence Profile

Application de gestion d’agence de transport développée avec Flask.

## Fonctionnalités

- Gestion des agences
- Gestion des trajets
- Gestion des villes
- Gestion des bus
- Gestion des places
- Gestion des réservations
- Validation complète des données
- Tests automatiques avec pytest
